# Jack Burrus

Full-stack engineer exploring AI and developer tooling. Previously built DeFi products in the Ethereum ecosystem.

**Stack:** TypeScript, React, Python, Rust, Node.js

## Projects

- **[agent-check](https://github.com/jackburrus/agent-check)** — A testing library for AI agents inspired by React Testing Library. Test agent behavior, not model outputs — verify tool calls, ordering, cost budgets, and policy compliance with full TypeScript support.

- **[snug-ai](https://github.com/jackburrus/snug-ai)** — A TypeScript library that intelligently packs context into an LLM's context window. Prioritizes content by importance, applies lost-in-the-middle placement based on attention research, and gives full visibility into what was included, dropped, and why.

- **[dotpack](https://dotpack.dev)** — A configuration package that gives AI coding tools (Claude Code, Cursor, Gemini CLI) pre-built agents, slash commands, and auto-detected skills across all your projects.

- **[ripdoc](https://github.com/jackburrus/ripdoc)** — Rust-native PDF extraction, 73x faster than pdfplumber. Drop-in replacement with PyO3 bindings — extract text, tables, words, and layout from PDFs at speeds that feel instant.

- **[speakhuman](https://github.com/jackburrus/speakhuman)** — A Rust-accelerated fork of humanize. Turns numbers into fuzzy human-readable durations, sizes, and throughput with localization support for 30+ languages.
